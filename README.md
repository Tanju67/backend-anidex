# 🌀 Anidex Backend

The modern, secure, and scalable **RESTful API** powering the **Anidex** ecosystem. Built with **TypeScript**, secured with **JWT**, and fully documented with **Swagger**.

> [!TIP]
> **Frontend Repository:** [Anidex Web](https://github.com/Tanju67/frontend-anidex.git)  
> **Live Preview:** [Anidex App](https://r-anime.onrender.com)

## ✨ Key Features

- **Robust Authentication:** Secure JWT-based auth and Google OAuth 2.0 integration.
- **Type Safety:** 100% TypeScript coverage for a reliable developer experience.
- **Zod Validation:** Strict schema validation for all incoming requests.
- **Interactive API Docs:** Fully detailed Swagger UI at `/api-docs`.
- **Watchlist Engine:** Efficient anime management with pagination and duplicate prevention.

## 🛠️ Tech Stack

- **Runtime:** Node.js (Express.js)
- **Language:** TypeScript
- **Database:** MongoDB (Mongoose ODM)
- **Security:** CORS, JWT
- **Documentation:** Swagger (OpenAPI 3.0)
- **Validation:** Zod
